# J.A.R.V.I.S.

A Windows desktop assistant built with Batch scripting for automating common tasks and providing quick access to system features.

## Overview

J.A.R.V.I.S. (Just A Rather Very Intelligent System) is a personal desktop automation tool inspired by the AI assistant from Iron Man. This lightweight Batch script provides voice-activated commands and system shortcuts for Windows environments.

## Features

- **System Automation**: Quick access to common Windows features and settings
- **User Management**: Simple user authentication system with users.txt
- **Hotspot Control**: Toggle Windows mobile hotspot on/off
- **Custom Icon**: Includes jarvis.ico for desktop shortcut personalization
- **Lightweight**: Built entirely with native Windows Batch scripting

## Project Structure

```
J.A.R.V.I.S./
├── J.A.R.V.I.S.bat         # Main executable script
├── jarvis.ico             # Custom icon file
├── programbase.dll        # Supporting library
├── programdata.dll        # Data management library
├── turn hotspot on.txt    # Hotspot control instructions
└── users.txt              # User credentials storage
```

## Usage

1. Double-click `J.A.R.V.I.S.bat` to launch the assistant
2. Follow the on-screen prompts for authentication
3. Use voice commands or menu options to execute tasks

## Requirements

- Windows OS (7/8/10/11)
- Administrative privileges for certain operations
